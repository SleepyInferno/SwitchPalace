# Phase 1 — UI Design Contract

> Visual and interaction contract for the Foundation & Install Pipeline phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (not a web project) |
| Preset | not applicable |
| Component library | borealis (xfangfang fork) — Nintendo Switch native UI framework |
| Icon library | borealis built-in icons + custom inline glyphs (checkmark, X, warning) |
| Font | borealis bundled Nintendo-adjacent sans-serif (framework default) |

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-text gaps, badge internal padding |
| sm | 8px | List item internal padding, inline element spacing |
| sm+ | 12px | Compact section padding, caption spacing below progress bar |
| md | 16px | Default element spacing, list item vertical padding |
| lg | 24px | Section padding, dialog internal padding |
| xl | 32px | Screen-level content margins |
| 2xl | 48px | Major section breaks between file list and action bar |
| 3xl | 64px | Page-level top/bottom padding |

Exceptions: Controller touch targets minimum 44px height for all focusable list items and buttons (accessibility on touchscreen mode).

---

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Screen Title | 34sp | 700 (Bold) | 1.2 |
| Section Header | 22sp | 600 (Semibold) | 1.2 |
| Body / List Item | 17sp | 400 (Regular) | 1.5 |
| Metadata / Caption | 13sp | 400 (Regular) | 1.4 |
| Badge / Label | 11sp | 600 (Semibold) | 1.0 |

Note: "sp" is borealis scale-independent pixels. Use borealis framework font at all sizes; no custom fonts.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#1A1A1A` Deep Charcoal | Full-screen background, all view backgrounds |
| Secondary (30%) | `#242424` Panel Dark / `#2E2E2E` Elevated | Surface cards (file list panel), dialogs, overlays, destination selector |
| Accent (10%) | `#3D9CF0` Royal Sapphire | Progress bar fill, selected list item highlight, primary action button fill, active tab indicator |
| Accent Hover | `#5DB0FF` Sapphire Bright | Focused button state (controller highlight), progress bar pulse on completion |
| Destructive | `#FF453A` Switch Red | Failed install X icon, error messages, cancel confirmation destructive action |
| Success | `#30D158` Switch Green | Completed install checkmark icon, hash verification pass indicator |
| Warning | `#FFD60A` Switch Yellow | Applet mode badge background |
| NAND Warning | `#FF9F0A` Caution Amber | NAND destination warning banner background |
| Text Primary | `#FFFFFF` | All body text, headings, list item filenames |
| Text Secondary | `#ADADAD` Silver | File sizes, metadata captions, speed/ETA readout |
| Text Disabled | `#5A5A5A` Dim | Grayed-out NSZ/XCZ items in applet mode |
| Border | `#3A3A3A` Subtle Line | List item separators, panel borders |

Accent reserved for: progress bar fill, selected/focused list item background, primary "Install" button, batch progress percentage text. Never used for body text, borders, or passive elements.

## Interaction Contract

### Controller Navigation

| Input | Context | Action |
|-------|---------|--------|
| D-pad Up/Down | File list | Move focus between file items |
| A button | File list (unfocused item) | Toggle selection (multi-select) |
| A button | "Install Selected" button focused | Begin install flow |
| B button | File list | Deselect all / exit app (if none selected) |
| B button | Install progress | No action (cancel requires explicit button focus) |
| B button | Summary screen | Return to file browser |
| L/R bumpers | Destination selector | Toggle between SD Card and NAND |
| A button | Cancel dialog: focused button | Confirm focused action |
| B button | Cancel dialog | Dismiss dialog (continue installing) |

### State Transitions

```
[File Browser] ---(select files + press Install)---> [Destination Confirm implicit in action bar]
[File Browser + action bar] ---(A on Install)---> [Install Progress]
[Install Progress] ---(all complete)---> [Post-Install Summary]
[Install Progress] ---(focus Cancel + A)---> [Cancel Confirmation Dialog]
[Cancel Confirmation] ---(confirm cancel)---> [Post-Install Summary with rollback note]
[Cancel Confirmation] ---(dismiss)---> [Install Progress]
[Post-Install Summary] ---(B)---> [File Browser]
[File Browser] ---(conflict detected before install)---> [Conflict Error Dialog]
[Conflict Error Dialog] ---(OK)---> [File Browser]
```
